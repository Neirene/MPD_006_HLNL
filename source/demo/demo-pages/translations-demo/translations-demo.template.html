Basic Translations

To use translations you can use the default translations, but they're only English.
The 'NotSpecified' translation: {{ 'NotSpecified' | raetUiTranslate }}

Different Language

To setup a different language we have to configure the `raetUiTranslationServiceProvider`.
To do so, you can use the translations provided by the `raetUiLibraryTranslations`. 

.config(['raetUiTranslationServiceProvider', 'raetUiLibraryTranslations', function (raetUiTranslationServiceProvider, raetUiLibraryTranslations) {
    // Set the default language to Dutch
    raetUiTranslationServiceProvider.translations = raetUiLibraryTranslations['nl-NL'];
}])

Override available translations

We can override a translation for a language.
To do so change the contents of the `key` from `raetUiLibraryTranslations['en-GB']['key']`

This is the 'ToBeOverriden' translation: `{{ 'ToBeOverriden' | raetUiTranslate }}` 

.config(['raetUiTranslationServiceProvider', 'raetUiLibraryTranslations', function (raetUiTranslationServiceProvider, raetUiLibraryTranslations) {
    // Override a translation from English
    raetUiLibraryTranslations['en-GB']['ToBeOverriden'] = 'The overrided translation';
    // Set the default language to English
    raetUiTranslationServiceProvider.translations = raetUiLibraryTranslations['en-GB'];
}])

Unavailable translation

If a translation isn't available, it will be preceded by a `#`.
The translation for 'NotTranslated' is `{{ 'NotTranslated' | raetUiTranslate }}`.

All translatable keys

Please be aware that ToBeOverriden is not part of the library but an example of overriding a translation.
